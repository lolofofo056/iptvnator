IPTVnator **v0.20** is out.

This release brings one of the largest UI updates so far. M3U, Xtream, and Stalker now share a more consistent application shell, the app opens around content instead of playlists, and everyday navigation is faster through the new dashboard, command palette, and scope-aware search.

It also includes important platform work: macOS builds are now signed, the Linux Flatpak package is fixed, Stalker Live TV gets EPG support, and several playback and playlist-management issues were cleaned up.

## Dashboard and navigation

<StatusPill type="new" />

- Added a new dashboard screen with basic widgets.
- Switched the default experience toward a content-first start instead of a playlist-first approach.
- Added settings to disable the dashboard when you prefer the classic flow.
- Added startup-view controls so IPTVnator can open the first available view or restore the last opened section-level view.
- Added a new command palette with `Cmd/Ctrl + K` for context-aware quick commands.
- Added scope-aware search that follows the current section.

## UI and settings

<StatusPill type="improved" />

- Reworked the settings page.
- The app now follows the system theme by default.
- Added UI customization options for dashboard visibility and the external player bar.
- Improved visual consistency across M3U, Xtream, and Stalker.
- Added a dedicated external player bar that can control an opened MPV/VLC player from inside IPTVnator.

## Favorites and history

<StatusPill type="improved" />

- Reworked Favorites and Recently Viewed across the app.
- Added channel sorting.
- Added drag-and-drop support.
- Integrated EPG context into favorites and history views.
- Improved consistency across all modules.

## Playlist management

<StatusPill type="improved" />

- Reworked the Add Playlist dialog into one unified flow with M3U, Xtream, and Stalker tabs.
- Added an Xtream action to clear the form.
- Fixed drag-and-drop import for M3U files from the file system.

## Search and discovery

<StatusPill type="improved" />

- Added consistent category search across modules.
- Added Xtream category search.
- Preserved hidden Xtream categories after refresh.
- Improved search behavior so it reflects the active playlist section.

## Module updates

<StatusPill type="new" />

### M3U

- Fully redesigned layout with a two-column categories and channels view.
- Replaced group accordions with a groups view.
- Improved sorting and category management.
- Added channel info dialog access from the channel context menu.

### Xtream

- Improved the account information dialog.

### Stalker

- Added EPG support for Live TV.
- Added channel sorting.
